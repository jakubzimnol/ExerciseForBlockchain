To build this project you need to install 
 ODB (version 2.4.0) https://www.codesynthesis.com/products/odb/,
 libmysqlclient (version 6.1.11) https://downloads.mysql.com/archives/c-c/,
 gRPC (version 3.13) https://grpc.io/docs/languages/cpp/quickstart/ - not working yet.
 
To run this app pass datbase connection arguments ie. --user <userName> --password <password> --host <host>  --database <databaseName> --port 3306
